## 🔧 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/StutiJain4999/Stuti-Portfolio.git
   cd Stuti-Portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build the production bundle**:
   ```bash
   npm run build
   ```
